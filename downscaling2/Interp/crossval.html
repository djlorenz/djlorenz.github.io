Interpolating to a Grid

Cross Validation

The parameters of the Kriging variogram are found via cross validation: a station is removed, Kriging is used to estimate the value at the removed station from all the other stations, and the error at the removed station is recorded. Next this process is repeated so that all stations have a chance to be removed. The parameters that lead to the least cross-validated error are used for the Kriging variogram.
